# HelpCleaner Documentation

This documentation provides an overview of **HelpCleaner**, its features, installation instructions, usage, logs, requirements, and licensing information.

---

## Overview

**HelpCleaner** is a lightweight Windows system utility designed to help users maintain, clean, and monitor their computer safely and efficiently. It combines file cleaning, virus scanning, and virtual drive management into a single, easy-to-use interface.

---

## Features

### System Virus Scan
- Scans your system for potential threats using the integrated virus scanner.
- Generates a detailed log of detected items.

### System Cleaning
- Scans for unnecessary or large files and prompts for confirmation before deletion.
- Helps free up storage space and improve system performance.

### Dangerous Files Management
- Lists potentially harmful or dangerous files on your system.
- Gives you the option to review and handle these files safely.

### Virtual Drive Management
- Create and mount virtual drives (VHD) to test or isolate files without affecting your main system.

### Storage Monitoring
- Displays storage usage for all available drives.
- Helps you track used space and manage storage efficiently.
